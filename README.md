# Expirysense

A Streamlit-based web application for managing and tracking expiry dates of products and items.

## Features

- User authentication (login/signup)
- Product expiry date management
- Search and filter functionality
- Modern, responsive UI with dark theme
- Data visualization and reporting

## Installation

1. Clone this repository:
```bash
git clone <your-github-repo-url>
cd Expirysense
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run basicss.py
```

## Dependencies

- streamlit
- pandas
- PIL (Pillow)
- rapidfuzz
- datetime

## Usage

1. Start the application using `streamlit run basicss.py`
2. Create an account or login with existing credentials
3. Add products with their expiry dates
4. Search and filter products
5. Monitor expiry dates and get notifications
